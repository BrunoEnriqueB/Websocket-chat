<h1>Simple websocket chat</h1>

<p>Projeto criado para estudar websocket puro, tanto no server side quanto no client side</p>

<h2>Proposta</h2>

<p>A proposta do projeto foi criar uma simples aplicação de chat usando serveless</p>

<h2>Rodando o projeto</h2>

<p>Para rodar o projeto, é necessario o Node.js, na sua versão 20 para cima</p>

<h3>Scripts</h3>

<ul>
  <li>
    npm start: Para rodar o projeto
  </li>
  <li>
    npm start:watch: Rode em um terminal de javascript debug para conseguir debugar a aplicação
  </li>
</ul>

<p>Acesse http://localhost:3000/chat para utilizar o chat da aplicação</p>
